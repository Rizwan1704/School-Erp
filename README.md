# 🏫 School ERP System

A simple and complete School ERP (Enterprise Resource Planning) system developed as a student project using HTML, CSS, and JavaScript. The system helps manage students, teachers, attendance, marks, fees, timetables, and notifications in a single platform.

## ✨ Features
- Student Management
- Teacher Management
- Attendance Tracking
- Marks & Exam Management
- Fee Collection System
- Timetable Management
- Notifications
- Role-Based Login (Admin, Teacher, Student)
- Local Storage Data Persistence

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser LocalStorage

## 🚀 How to Run
1. Download the project files.
2. Open `index.html` in any web browser.
3. Login with demo credentials and explore the system.

## 🎯 Project Objective
To provide a digital solution for managing school operations efficiently while demonstrating web development skills and ERP concepts.

## 📚 Learning Outcomes
- Frontend Development
- JavaScript Programming
- CRUD Operations
- Role-Based Access Control
- Data Management using LocalStorage
